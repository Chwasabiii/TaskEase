# TaskEase

Accessible Task Management System built with Vite + React + Tailwind.

## Features

- Keyboard-first task management
- High-contrast and text scaling controls
- Focus mode and reminder support
- User login/signup with local persistence
- Task steps/subtasks and archive history
- Mobile-friendly UI with animated login preview

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
